ESP32-S3

PCA9555 / PCAL9555A

Both chips provide 16 GPIO pins in two 8-bit ports over I2C. The driver auto-detects which variant is connected and enables features accordingly:

Feature PCA9555 PCAL9555A
16 GPIO, I2C, polarity inversion Yes Yes
Pull-up / pull-down resistors ‚Äì Yes
Drive strength, input latch ‚Äì Yes
Interrupt mask / status ‚Äì Yes
Output mode (push-pull / OD) ‚Äì Yes

Interactive Input Test

Disabled by default. Requires a momentary push-button between PCA9555 pin 0 and GND:

  • PCAL9555A: Internal pull-up is enabled automatically
  • PCA9555: Add an external 10k pull-up resistor to VDD

The test waits 10 seconds for a button press, logging a countdown. If no button is connected, it times out gracefully without failing.

LED Animation Demo

File: main/pcal95555_led_animation.cpp

Drives 16 LEDs through 10 animation patterns to visually test the driver‚Äôs output capabilities and I2C throughput. Uses only PCA9555-compatible registers, so it works with both chip variants.

Animation Patterns

  1. Sequential Chase ‚Äì single LED scans left to right and back
  2. Bounce ‚Äì LED bounces between endpoints with acceleration
  3. Binary Counter ‚Äì counts 0-65535 showing binary on 16 LEDs
  4. Breathing (PWM) ‚Äì software PWM fade-in/fade-out of all LEDs
  5. Wave / Comet Tail ‚Äì 4-LED comet sweeps back and forth
  6. Random Sparkle ‚Äì random LED patterns at fast rate
  7. Build-up / Teardown ‚Äì LEDs light sequentially then extinguish
  8. Accelerating Scan ‚Äì speed ramps from 120ms down to 1ms and back
  9. Center Expand ‚Äì symmetric outward/inward animation from center
  10. Alternating Flash ‚Äì port-vs-port and even-vs-odd flashing

LED Wiring

Connect LEDs with 220-1k current-limiting resistors to each I/O pin.

  • Active-HIGH (default): LED anode to IO pin, cathode to GND
  • Active-LOW: Set LEDS_ACTIVE_LOW = true in the source, LED cathode to IO pin, anode to VDD

CONFIG_ESP_MAIN_TASK_STACK_SIZE=16384

Troubleshooting

I2C NACK Errors

Symptom Likely Cause Fix
NACK on all registers Wrong address / no power Check wiring, verify address
NACK on 0x40-0x4F only PCA9555 (not PCAL9555A) Expected ‚Äì driver auto-detects
Intermittent NACKs Bus noise, missing pull-ups Add 4.7k pull-ups

Stack Overflow / Assert 

1

assert failed: xTaskRemoveFromEventList tasks.c

Increase stack size: CONFIG_ESP_MAIN_TASK_STACK_SIZE=16384 in sdkconfig.

ESP32-S3 Flash Failure 

1

Manually enter download mode (BOOT + RESET) before flashing.
